## How It Works

- **When:** 2nd Tuesday monthly, 7pm Pacific
- **Game:** No-limit Texas Hold'em
- **Buy-in:** $50, up to 2 rebuys in the first hour (cutoff 8:05pm PT)
- **Payouts:** pool > $500 → 50 / 30 / 20 (three spots); otherwise 70 / 30 (two spots)
